# Анализ задачи

Задача — реализовать модуль `aiPlayer` для игры крестики-нолики. Проект уже имеет структуру директорий и большинство файлов. Нужно реализовать два файла:
- `src/logic/aiPlayer.ts` — логика minimax
- `src/logic/aiPlayer.test.ts` — тесты

Оба файла уже существуют в дереве проекта (судя по листингу), но требуют реализации. Зависимости: `src/types/index.ts` и `src/logic/gameLogic.ts` уже существуют. Это задача на **1 стадию** — оба файла логически связаны и реализуются вместе.
