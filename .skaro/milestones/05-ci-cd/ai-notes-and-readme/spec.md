# Specification: ai-notes-and-readme

## Context
Согласно правилам LLM в конституции проекта, необходимо создать `AI_NOTES.md`. Также необходим актуальный `README.md` для разработчиков и пользователей репозитория. Документация завершает проект и делает его понятным для внешних участников.

## User Scenarios
1. **Новый разработчик клонирует репозиторий:** читает `README.md` и за 5 минут запускает проект локально.
2. **Ревьюер проверяет AI_NOTES.md:** понимает архитектурные решения и компромиссы, принятые при разработке.

## Functional Requirements
- FR-01: Создан `AI_NOTES.md` по шаблону проекта: описание принятых решений, отклонений от спецификации (если есть), известных ограничений.
- FR-02: `README.md` содержит: название проекта, краткое описание, ссылку на задеплоенное приложение, инструкции `npm install`, `npm run dev`, `npm test`, `npm run build`.
- FR-03: `README.md` содержит описание режимов игры (PvP и PvAI).
- FR-04: `README.md` содержит описание технологического стека.
- FR-05: `README.md` содержит информацию о структуре проекта (краткое описание папок).
- FR-06: Все команды в `README.md` проверены и работают корректно.

## Non-Functional Requirements
- NFR-01: `README.md` написан на русском языке (согласно языковым правилам проекта).
- NFR-02: `AI_NOTES.md` написан на русском языке.

## Boundaries (что НЕ входит)
- Не создаётся полная документация API.
- Не создаётся CHANGELOG.
- Не создаётся CONTRIBUTING.md.

## Acceptance Criteria
- [ ] `AI_NOTES.md` создан и содержит описание ключевых решений.
- [ ] `README.md` содержит ссылку на задеплоенное приложение.
- [ ] `README.md` содержит все необходимые команды для запуска и тестирования.
- [ ] Все команды в `README.md` проверены и работают.
- [ ] Оба файла написаны на русском языке.
