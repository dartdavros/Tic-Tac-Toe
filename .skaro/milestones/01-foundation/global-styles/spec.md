# Specification: global-styles

## Context
Все компоненты используют CSS-переменные из `global.css` как единственный источник токенов дизайна. Без этого файла компоненты не смогут использовать переменные цветов, размеров и шрифтов. Согласно архитектурному инварианту №15, хардкод значений в `*.module.css` запрещён.

## User Scenarios
1. **Разработчик создаёт компонент:** использует `var(--color-player-x)` в CSS-модуле — браузер корректно применяет цвет `#e74c3c`.
2. **Пользователь открывает приложение на мобильном:** шрифты и отступы корректно масштабируются согласно медиазапросам.

## Functional Requirements
- FR-01: Создан файл `src/styles/global.css` и подключён в `src/main.tsx` через `import './styles/global.css'`.
- FR-02: Определены CSS-переменные цветов в `:root`: `--color-player-x: #e74c3c`, `--color-player-o: #3498db`, `--color-bg: #f9f9f9`, `--color-surface: #ffffff`, `--color-border: #cccccc`, `--color-text-primary: #222222`, `--color-text-secondary: #666666`, `--color-accent: #27ae60`, `--color-win-highlight: #f1c40f`.
- FR-03: Определены переменные размеров клетки в `:root`: `--cell-size-desktop: 120px` применяется по умолчанию как основной размер клетки. В медиазапросе `@media (max-width: 480px)` переменная `--cell-size-desktop` переопределяется значением `--cell-size-mobile` (`80px`), тем самым адаптируя размер клетки для мобильных устройств без необходимости менять ссылки в компонентах.
- FR-04: Определены переменные типографики в `:root`: `--font-family: 'Segoe UI', system-ui, sans-serif`, `--font-size-base: 16px`, `--font-size-large: 24px`.
- FR-05: Применён CSS-сброс: `box-sizing: border-box` для всех элементов (`*, *::before, *::after`), `margin: 0` и `padding: 0` для `body`.
- FR-06: Тег `body` использует `font-family: var(--font-family)`, `font-size: var(--font-size-base)`, `background-color: var(--color-bg)`, `color: var(--color-text-primary)`.
- FR-07: Добавлен медиазапрос `@media (max-width: 480px)`, в котором переопределяются следующие переменные: `--font-size-base: 14px`, `--font-size-large: 18px`, `--cell-size-desktop: 80px` (тем самым применяя значение мобильного размера клетки).
- FR-08: Приложение корректно отображается при ширине экрана от 320px без горизонтального скролла.
- FR-09: Определена переменная скругления углов в `:root`: `--border-radius: 8px`.
- FR-10: Определена переменная токена отступа в `:root`: `--spacing-base: 16px`.

## Non-Functional Requirements
- NFR-01: Файл не содержит компонентно-специфичных стилей — только глобальные сбросы и переменные.
- NFR-02: Все цвета, размеры и шрифты определены исключительно через CSS-переменные в `:root`.

## Полный список CSS-переменных `:root`

| Переменная | Значение | Назначение |
|---|---|---|
| `--color-player-x` | `#e74c3c` | Цвет игрока X |
| `--color-player-o` | `#3498db` | Цвет игрока O |
| `--color-bg` | `#f9f9f9` | Цвет фона страницы |
| `--color-surface` | `#ffffff` | Цвет поверхности карточек |
| `--color-border` | `#cccccc` | Цвет границ |
| `--color-text-primary` | `#222222` | Основной цвет текста |
| `--color-text-secondary` | `#666666` | Вторичный цвет текста |
| `--color-accent` | `#27ae60` | Акцентный цвет |
| `--color-win-highlight` | `#f1c40f` | Цвет подсветки победных клеток |
| `--cell-size-desktop` | `120px` | Размер клетки (десктоп и по умолчанию) |
| `--cell-size-mobile` | `80px` | Размер клетки (мобильные, справочное значение) |
| `--font-family` | `'Segoe UI', system-ui, sans-serif` | Семейство шрифтов |
| `--font-size-base` | `16px` | Базовый размер шрифта |
| `--font-size-large` | `24px` | Крупный размер шрифта |
| `--border-radius` | `8px` | Радиус скругления углов |
| `--spacing-base` | `16px` | Базовый токен отступа |

## Переопределения в `@media (max-width: 480px)`

| Переменная | Значение в медиазапросе | Обоснование |
|---|---|---|
| `--font-size-base` | `14px` | Уменьшение базового шрифта для малых экранов |
| `--font-size-large` | `18px` | Уменьшение крупного шрифта для малых экранов |
| `--cell-size-desktop` | `80px` | Применение мобильного размера клетки без смены ссылок в компонентах |

## Boundaries (что НЕ входит)
- Не создаются CSS-модули компонентов (это делается в задачах соответствующих компонентов).
- Не реализуется тёмная тема.
- Не подключаются внешние шрифты через `@import` (используются системные шрифты).
- Не добавляются дополнительные токены отступов помимо `--spacing-base`.

## Acceptance Criteria
- [ ] Файл `src/styles/global.css` создан и содержит все 16 CSS-переменных из таблицы выше.
- [ ] `import './styles/global.css'` присутствует в `src/main.tsx`.
- [ ] В браузере (DevTools → Elements → :root) видны все CSS-переменные, включая `--color-win-highlight`, `--border-radius` и `--spacing-base`.
- [ ] Страница корректно отображается на ширине 320px без горизонтального скролла.
- [ ] При ширине экрана ≤ 480px переменные `--font-size-base`, `--font-size-large` и `--cell-size-desktop` принимают мобильные значения (проверяется через DevTools → Computed).
- [ ] При ширине экрана > 480px переменная `--cell-size-desktop` равна `120px`.
- [ ] `npm run build` проходит без ошибок.
