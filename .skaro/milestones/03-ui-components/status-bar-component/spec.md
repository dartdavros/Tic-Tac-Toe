# Specification: status-bar-component

## Context
Компонент `StatusBar` информирует пользователя о текущем состоянии игры: чей ход, кто победил или что наступила ничья. Согласно ADR-010, компонент использует `aria-live="polite"` для объявления изменений скринридерам. Компонент является чисто презентационным.

## User Scenarios
1. **Игра в процессе:** отображается «Ход игрока X» или «Ход игрока O».
2. **Победа:** отображается «Победил игрок X!» или «Победил игрок O!».
3. **Ничья:** отображается «Ничья!».
4. **Скринридер:** при смене статуса объявляется новое сообщение через `aria-live`.

## Functional Requirements
- FR-01: Компонент принимает props: `status: GameStatus`.
- FR-02: При `status.kind === 'playing'` отображает текст «Ход игрока {currentPlayer}».
- FR-03: При `status.kind === 'won'` отображает текст «Победил игрок {winner}!».
- FR-04: При `status.kind === 'draw'` отображает текст «Ничья!».
- FR-05: Корневой элемент имеет атрибут `aria-live="polite"`.
- FR-06: Цвет текста для X — `var(--color-player-x)`, для O — `var(--color-player-o)`, для ничьей — `var(--color-text-secondary)`.
- FR-07: Стили определены в `src/styles/StatusBar.module.css`.
- FR-08: Создан `src/components/StatusBar.test.tsx` со smoke-тестами для каждого `status.kind`.

## Non-Functional Requirements
- NFR-01: Компонент не хранит локального состояния.
- NFR-02: Текст статуса понятен без визуального контекста (для скринридеров).

## Boundaries (что НЕ входит)
- Не реализуется таймер или счётчик ходов.
- Не реализуется история ходов.
- Не реализуется анимация смены статуса.

## Acceptance Criteria
- [ ] При `status.kind === 'playing'` отображается корректный текст с именем текущего игрока.
- [ ] При `status.kind === 'won'` отображается имя победителя.
- [ ] При `status.kind === 'draw'` отображается «Ничья!».
- [ ] Присутствует `aria-live="polite"` на корневом элементе.
- [ ] Smoke-тесты в `StatusBar.test.tsx` проходят для всех трёх `status.kind`.
- [ ] `npm run lint` не выдаёт ошибок.

## Open Questions
- Нужно ли добавить `aria-atomic="true"` для объявления всего текста целиком при изменении?
- Стоит ли добавить иконку или эмодзи рядом с именем игрока для визуального различия?
