# Specification: game-settings-component

## Context
Компонент `GameSettings` является начальным экраном приложения. Пользователь выбирает режим игры и, в режиме PvAI, свою сторону. После подтверждения вызывается `onStartGame` с объектом `GameSettings`. Согласно ADR-010, все элементы формы имеют явные `<label>`.

## User Scenarios
1. **Пользователь выбирает PvP:** нажимает «Начать игру» — вызывается `onStartGame({ mode: 'pvp' })`.
2. **Пользователь выбирает PvAI и сторону X:** нажимает «Начать игру» — вызывается `onStartGame({ mode: 'pvai', humanPlayer: 'X' })`.
3. **Пользователь выбирает PvAI и сторону O:** нажимает «Начать игру» — вызывается `onStartGame({ mode: 'pvai', humanPlayer: 'O' })`.
4. **По умолчанию:** при открытии меню выбран режим PvP.

## Functional Requirements
- FR-01: Компонент принимает props: `onStartGame: (settings: GameSettings) => void`.
- FR-02: Отображает заголовок «Крестики-нолики».
- FR-03: Отображает выбор режима: две радиокнопки «Два игрока» (PvP) и «Против компьютера» (PvAI) с явными `<label>`.
- FR-04: При выборе режима PvAI отображается дополнительный выбор стороны: две радиокнопки «Играть за X» и «Играть за O» с явными `<label>`.
- FR-05: При выборе режима PvP блок выбора стороны скрывается.
- FR-06: Кнопка «Начать игру» вызывает `onStartGame` с корректным объектом `GameSettings`.
- FR-07: По умолчанию выбран режим PvP; в режиме PvAI по умолчанию выбрана сторона X.
- FR-08: Все `<input>` имеют явные `<label>` (связанные через `htmlFor` / `id`).
- FR-09: Стили определены в `src/styles/GameSettings.module.css`.
- FR-10: Создан `src/components/GameSettings.test.tsx` со smoke-тестами.

## Non-Functional Requirements
- NFR-01: Компонент использует локальный `useState` только для управления состоянием формы (выбранный режим и сторона).
- NFR-02: Форма корректно отображается на ширине от 320px.

## Boundaries (что НЕ входит)
- Не реализуется валидация формы (все варианты всегда валидны).
- Не реализуется выбор уровня сложности ИИ.
- Не реализуется история предыдущих игр.

## Acceptance Criteria
- [ ] При выборе PvP и нажатии «Начать игру» вызывается `onStartGame({ mode: 'pvp' })`.
- [ ] При выборе PvAI + X вызывается `onStartGame({ mode: 'pvai', humanPlayer: 'X' })`.
- [ ] При выборе PvAI + O вызывается `onStartGame({ mode: 'pvai', humanPlayer: 'O' })`.
- [ ] Блок выбора стороны отображается только при выборе PvAI.
- [ ] Все `<input>` имеют связанные `<label>`.
- [ ] Smoke-тесты в `GameSettings.test.tsx` проходят.
- [ ] `npm run lint` не выдаёт ошибок.

## Open Questions

- Нужно ли добавить краткое описание каждого режима для пользователя?
