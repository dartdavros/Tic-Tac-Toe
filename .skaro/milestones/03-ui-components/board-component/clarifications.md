# Clarifications: board-component

## Question 1
Как визуально выделять клетки победной линии (winLine)?

*Context:* FR-05 требует дополнительный CSS-класс, но не уточняет визуальное оформление — это влияет на структуру CSS Modules и токены дизайна в global.css.

**Options:**
- A) Изменить фон клетки через CSS-переменную --color-win-highlight (например, #f1c40f)
- B) Изменить цвет символа внутри клетки и добавить рамку
- C) Добавить SVG-линию поверх поля как отдельный элемент (но это противоречит границам спецификации)
- D) Решение отдаётся компоненту Cell через проп isWinner, Board только передаёт флаг

**Answer:**
Изменить фон клетки через CSS-переменную --color-win-highlight (например, #f1c40f)

## Question 2

*Context:* FR-04 говорит, что занятая клетка получает disabled={true}, но не уточняет — должен ли Board дополнительно фильтровать вызов onCellClick или Cell сама блокирует событие.

**Options:**
- A) Board передаёт disabled={true} в Cell, Cell полностью блокирует событие — onCellClick не вызывается никогда для занятой клетки
- B) Board передаёт обработчик, который проверяет board[index] !== null и не вызывает onCellClick
- C) Ответственность только на Cell: Board всегда передаёт onCellClick, Cell игнорирует клик если disabled

**Answer:**
Board передаёт disabled={true} в Cell, Cell полностью блокирует событие — onCellClick не вызывается никогда для занятой клетки

## Question 3
Нужно ли компоненту Board добавлять role="row" и role="gridcell" для вложенных элементов согласно ADR-010?

*Context:* ADR-010 требует role="grid" на контейнере, но стандарт ARIA для grid предполагает вложенные role="row" и role="gridcell" — без них скринридеры могут некорректно интерпретировать структуру.

**Options:**
- A) Добавить три обёртки div с role="row", каждая содержит 3 Cell с role="gridcell" (полный ARIA grid)
- B) Ограничиться только role="grid" на контейнере, role="button" на Cell — минимальный a11y согласно ADR-010
- C) Добавить role="row" без дополнительных обёрток (невалидно, но проще)

**Answer:**
Добавить три обёртки div с role="row", каждая содержит 3 Cell с role="gridcell" (полный ARIA grid)

## Question 4

**Options:**
- A) gap в CSS Grid на Board (чистое разделение, без border на Cell)
- B) border на Cell (классический вид крестиков-ноликов с линиями сетки)
- C) Комбинация: gap=0 и border только на нужных сторонах Cell для имитации линий сетки
